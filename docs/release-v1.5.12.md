# ARUN v1.5.12 Release Notes

ARUN v1.5.12 is a patch release for implementation-heavy scrum product
coherence after evaluating the `hakobune8/arun-test` branch generated by
`run-7c8726d62db61ea9`.

## Changed

- Strengthened the built-in `implementation-heavy-scrum` template so Sprint 1
  produces one concise source-of-truth product brief instead of competing
  product names, mechanics, or documentation narratives.
- Required frontend, backend, README, docs, app title, visible labels, and
  served root behavior to match the same product concept.
- Marked concept drift, generic scaffold output, unserved alternate UI trees,
  placeholder root responses, broken docs links, and missing differentiating
  mechanics as release-blocking QA and review gaps.
- Updated the Web UI workspace label, Helm chart `version`, chart
  `appVersion`, and default image tag for v1.5.12.

## Upgrade

Use image tag `ghcr.io/hakobune8/arun:v1.5.12` and Helm chart version `1.5.12`.

```bash
helm upgrade arun oci://ghcr.io/hakobune8/charts/arun \
  --version 1.5.12 \
  --namespace arun \
  --reuse-values \
  --set image.tag=v1.5.12
```
